
## リストの値を集計する

sum関数を使います

### フォーマット

    対象のリスト.sum()

### 実装サンプル

    // Array<Int>のインスタンス生成
    val array = arrayOf(1,2,3,4,5)

    // sumする(1 + 2 + 3 + 4 + 5)
    println(array.sum()) // => 15

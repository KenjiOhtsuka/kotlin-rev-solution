
# Kotlinのキーワードを知りたい

KotlinではJavaとのcompatibilityを目的にしたものはアノテーション、Kotlin側で使われるものをキーワードと表現する傾向があります。

キーワードには以下のようなものがあります。

## public

キーワードというかアクセス修飾子。
Javaとほぼ同じ。Kotlinではデフォルトのアクセスレベルがpublicとなっている。

## private

キーワードというかアクセス修飾子。
Javaとほぼ同じ。


## open

classの継承をする際に親クラスにつける。
Kotlinではデフォルトで継承不可なクラスとなっている。

    open class Hoge
